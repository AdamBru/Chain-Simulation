# Chain Simulation

Symulator łańcucha napisany w Javie. Posiada prosty silnik fizyczny oraz możliwość zmiany parametrów symulacji. Przez przeciąganie ogniw łańcucha można nim poruszać. 

> ***Projekt korzysta i wymaga JavaFX.***
